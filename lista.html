Lista

Uma Lista é uma estrutura de dados que armazena uma coleção de elementos em sequência, podendo ser acessados individualmente através de um índice. Cada elemento da Lista é identificado por um número inteiro que representa sua posição na sequência.

Os principais métodos de uma Lista são:

  • Append: Adiciona um elemento ao final da Lista
  • Insert: Adiciona um elemento em uma posição específica da Lista
  • Remove: Remove um elemento da Lista
  • Get: Retorna um elemento da Lista em uma posição específica
  • Size: Retorna o número de elementos na Lista

As motivações para o uso de Listas são diversas, como em algoritmos que precisam armazenar e acessar uma sequência de elementos. Ela pode ser utilizada para armazenar informações em um banco de dados, por exemplo.

A primeira vez que a estrutura de dados Lista foi utilizada foi em 1951, pelo matemático americano John Mauchly.
